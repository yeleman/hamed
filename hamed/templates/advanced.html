Mode avancé

Le mode avancé vous permet d'utiliser certaines fonctionnalités destructrices de données. Il est désactivé par défaut.

Sur demande auprès de la DNDS, il est possible de l'activer pour la journée. Cela pourra vous permettre de supprimer certaines données non voulues.

Contactez la DNDS en indiquant le code {{ request_code }} et renseignez le code qu'ils vous auront communiqué si dessous.

Activation

Après avoir reçu le code d'activation (ActivationCode) de la DNDS, renseignez-le ci-dessous, avec le code de demande.

Attention, vous devez indiquer comme RequestCode le code que vous avez fourni à la DNDS, et pas forcément celui affiché actuellement.
